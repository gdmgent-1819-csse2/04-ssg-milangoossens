Opdracht 04: Static Site Generator

Genereer met een Static Site Generator een documentatiewebsite die op GitHub Pages gehost wordt.

Maak een nieuwe repo op https://classroom.github.com/a/aaMf5TE9 en stel de repo in om GitHub Pages te gebruiken
(Settings â†’ GitHub Pages: master branch of master branch /docs folder).

Kies een SSG op https://www.staticgen.com/ en genereer lokaal de documentatiewebsite die je gaat pushen naar GitHub en hosten op GitHub Pages.

Maak een 5-tal voorbeeldpagina's in Markdown met tekst en codevoorbeelden (niet als afbeelding, maar als code). Baseer de voorbeelden in de documentatie op de vorige opdrachten. In een grafische opleiding wordt ook een verzorgde, eigen opmaak verwacht.
